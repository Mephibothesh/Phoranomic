从关系协调到时空对称性
论文标题
《从关系协调到时空对称性：狭义相对论的一种简单公理化》

核心思想
大多数教科书从“光速不变”出发推导狭义相对论。我认为这颠倒了因果。

真正的基础问题是：多个理性观察者如何协调他们对同一物理现实的描述？

三个公理
线性性（描述的一致性）

因果同构（逻辑的一致性）

固有时标量性（经验的一致性）

主要结论
从这三个公理可以必然推导出洛伦兹对称性（即狭义相对论的核心数学结构）。

这意味着：当不同的观察者必须共享一个可理解的物理世界时，闵可夫斯基几何就是他们必须同意的通信协议。

哲学意义
这是 “关系先于存在” 的一个数学例证：
时空结构并非先验存在，而是从观察者之间协调关系的必要性中涌现出来的。
